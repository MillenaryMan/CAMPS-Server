> [!IMPORTANT]
> Before you start, ensure you have a Linux-based virtual or dedicated server.
>
> Minimum system requirements for launching the server build:
> - OS: Linux-based (Ubuntu compatibility verified)
>
> - CPU: 2 Cores
>
> - RAM: 3GB 
>
> - Storage: 2GB
